## 📌 Get Started

1. Initialized (don't forget to rename `my-project`)

    ```bash
    npx degit owlsome-official/next-tailwind-ts#main my-project
    ```

2. Go to project folder

    ```bash
    cd my-project
    ```

3. Set up dependencies

    ```bash
    yarn
    ```

4. Run!

    ```bash
    yarn dev
    ```
